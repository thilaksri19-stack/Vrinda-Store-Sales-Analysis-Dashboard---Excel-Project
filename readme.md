# Vrinda Store Sales Analysis Dashboard

## Project Overview

This is my first Data Analytics project created using Microsoft Excel. The objective of this project is to analyze sales data from Vrinda Store and generate meaningful business insights through an interactive dashboard.

## Objectives

* Analyze monthly sales performance
* Compare sales between men and women customers
* Evaluate order status distribution
* Identify top-performing states
* Analyze sales by age group and gender
* Understand sales contribution from different channels
* Study category-wise product performance

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Creation
* Data Visualization

## Key Insights

* Women customers contributed more sales compared to men.
* Amazon was one of the top sales channels.
* Maharashtra generated the highest sales revenue.
* Most orders were successfully delivered.
* Adult age group contributed significantly to sales.

## Dashboard Features

* Orders vs Sales Analysis
* Gender-wise Sales Distribution
* Order Status Analysis
* Top 10 States by Revenue
* Age Group vs Gender Analysis
* Sales Channel Performance
* Product Category Analysis

## Dataset Information

The dataset contains more than 31,000 customer order records with information such as:

* Customer Details
* Product Category
* Sales Amount
* Order Status
* Sales Channel
* State Information
* Age Group
